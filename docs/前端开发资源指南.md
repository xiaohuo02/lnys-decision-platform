### 统一响应结构

所有后端接口返回统一格式：

```json
{
  "code": 200,
  "data": { ... },
  "message": "ok",
  "meta": {
    "degraded": false,
    "source": "csv",       // "db" | "csv" | "redis" | "mock" | "agent"
    "trace_id": "uuid...",
    "warnings": []
  }
}
```

**关键约定**：
- `meta.degraded = true` 表示返回的是 mock/降级数据，前端应显示降级标识
- `meta.source` 标识数据来源，可用于 debug 和展示
- `meta.trace_id` 可关联到 Console → Traces 页面查看执行详情
- 分页接口内层结构：`{ items: [...], total: N, page: 1, page_size: 20 }`

### 数据降级三级策略（GET 接口）
```
1. Redis 缓存命中 → 直接返回 ok(data)     [source: "redis"]
2. CSV 文件命中   → 读取、回写缓存 → ok    [source: "csv"]
3. Mock 降级      → ENABLE_MOCK_DATA=true  [source: "mock", degraded: true]
4. 全部失败       → 503 error
```

### POST 接口降级策略
```
1. Redis 缓存命中 → 返回 cached(data)
2. Agent 调用     → agent 就绪则调用 agent.run()
3. Agent 不可用   → Service 降级层（ML模型/规则引擎）
4. 全部失败       → 返回 mock 结果
```

## 6. SSE 实时推送

> **最重要的技术力展示点之一**

### 6.1 Workflow 进度推送

**连接**：`GET /api/v1/workflows/{run_id}/stream`

```javascript
const es = new EventSource(`/api/v1/workflows/${runId}/stream`)

es.addEventListener('route_decided', (e) => {
  // SupervisorAgent 路由决策完成
  const data = JSON.parse(e.data)
})

es.addEventListener('step_started', (e) => {
  // 某节点开始执行（如 customer_agent 开始分析）
})

es.addEventListener('step_completed', (e) => {
  // 某节点执行完成（附摘要和 artifact_id）
})

es.addEventListener('step_failed', (e) => {
  // 某节点执行失败
})

es.addEventListener('hitl_required', (e) => {
  // 需要人工审核 → 可跳转 HITL 审核页面
})

es.addEventListener('hitl_resolved', (e) => {
  // 人工审核完成 → workflow 继续
})

es.addEventListener('final', (e) => {
  // workflow 完成
  es.close()
})

es.addEventListener('error', (e) => {
  // workflow 失败
  es.close()
})
```

### 6.2 聊天流式推送

**连接**：`POST /api/chat/stream`（非 EventSource，需要 fetch + ReadableStream）

```javascript
const response = await fetch('/api/chat/stream', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ session_id, message, customer_id }),
})

const reader = response.body.getReader()
const decoder = new TextDecoder()

while (true) {
  const { done, value } = await reader.read()
  if (done) break
  const text = decoder.decode(value)
  // 解析 SSE 格式: "event: token\ndata: {\"content\":\"你\"}\n\n"
}
```
